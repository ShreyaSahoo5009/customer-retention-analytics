# Decoding Customer Value: A Data Driven Retention Strategy

## Overview
This project analyzes customer purchasing behavior, loyalty patterns, promotional dependency, and revenue trends using Python, SQL, and Power BI.

## Tech Stack
- Python
- Pandas
- NumPy
- SQL
- MySQL
- Power BI

## Key Features
- Customer 360 dataset creation
- Customer segmentation
- Loyalty score engineering
- Satisfaction analysis
- Revenue & seasonal analysis
- Geographic opportunity analysis
- Promotional dependency analysis

## Dashboard Insights
- Customer Value Pyramid
- Promotion Reliance vs Loyalty
- Geographic Opportunity Map
- Product Categories Driving Loyalty
- Seasonal & Shipping Analysis

## Files Included
- customer_analysis.py
- analysis_queries.sql
- customer_360.csv
- Power BI dashboard (.pbix)
- SUMMARY (.pdf)
- PLAYBOOK (.pdf)
- Dataset.csv
- cleaned_shopping_data.csv

## Business Outcome
Designed retention and promotional optimization strategies using customer analytics and business intelligence.
